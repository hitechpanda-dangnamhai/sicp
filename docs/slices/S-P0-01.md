# S-P0-01 — Multi-tenant SaaS (RLS + tenant_id)

## Goal
Mọi data path scope theo tenant, enforcement 3 lớp: RLS (DB) + app-level
helper + Vespa/Redis scope. Đóng BACKLOG §3 P0 #1; unblock #2 payment.

## Design (chỉ trỏ, không lặp body)
ADR-040 (+ amendment 2026-06-11: icp_app/SET LOCAL, 2 conn string, Vespa
helper, Redis prefix; + amendment b: NULLIF hardening) · ADR-046
(+ amendment: active-tenant JWT) · ADR-047 (header transport — code align
theo doc; + amendment 2026-06-12: header contract áp MỌI caller của MCP,
gồm Gateway direct tool call; MCP fail-closed sau T03) · ADR-041 (không
log PII).

## Evidence (RECON 2026-06-11)
- `icp` role superuser+BYPASSRLS; pg pool max=10 (gateway), psycopg
  conn-per-query (MCP); không pooler.
- 0 tenant_id / 17 bảng (FACTS); FK→users 8 bảng; products.merchant_id có sẵn.
- JWT thiếu tenant claim; Gateway→AI dùng JSON body user_id (lệch ADR-047).
- UNIQUE global vỡ multi-tenant: orders.idempotency_key, policies.code.
- behavior_events partitioned PK(event_id, occurred_at) — thêm cột, không đổi PK.
- 3 matview V006 GROUP BY thiếu tenant; trigger trg_review_update_product.
- Data thật: orders 1281, products 75, users 5 → backfill tenant `demo`.
- Vespa product.sd có merchant_id, YQL builder (vespa.py:147) không filter.
- Redis: idem:cache scope user; sse:pubsub + RedisSaver thread_id chưa scope.

  products.controller (×2: products.update + vespa.index),

  (dòng 114, di sản S-05 trước khi McpClient ra đời S-07), 7 tool cart.*.
  Report STOP cũ nói cart dùng mcp.client → SAI.
- Verified: KHÔNG có call-site MCP nào ngoài HTTP request flow

  abort-timeout + SSE heartbeat. KHÔNG có đường raw-fetch /rpc thứ 3.

- cart Redis key `cart:{user}`: CHỈ MCP chạm (Gateway đi qua cart.* tool,

## Stop conditions
- Phát hiện pooler transaction-mode trên đường runtime → STOP (SET LOCAL
  semantics vỡ), quay lại PLAN.
- Bất kỳ query/tool không thể đi qua helper → STOP, không bypass tay.
- Code path nào đòi tenant_id trên users/sessions → STOP (vi phạm ADR-046).
- Vespa backfill đòi re-feed mất embedding → STOP (ADR-036 Locked).
- Call-site MCP nào nằm NGOÀI HTTP request context (cron/seed/worker) —
  không có tenant để gửi header → STOP, báo PLAN xếp đường riêng.
- Task nào diff > ~1500 dòng → cắt đôi, báo lại PLAN.
