# ADR-057 — Coverage gate đo SOURCE-only (exclude spec files)

Status: Accepted
Liên quan: W-76 (coverage ratchet, S-P0-03/T01·T01b) · ADR-052

## Context
- T01b set floor gateway=43 với coverage `all:true include:['src/**']` KHÔNG exclude `*.spec.ts`
  → spec files VÀO denominator (floors.json comment: 23 spec WITH-DB = 43.8%).
- T02a thêm 1 spec (seed-foundation, src/database/) → ratio 43.8→42.72 < floor → CI integration-db
  đỏ. Thêm test LÀM TỤT coverage-gate = đo sai đối tượng (coverage của file test, không phải source).

## Decision
- Coverage gate đo SOURCE-only: exclude `**/*.spec.ts` (+ test helper) khỏi coverage include/
  denominator. Giữ `all:true` (đếm src chưa test = honest floor).
- Re-measure floor trên basis source-only → set floors.json mới (số honest, có thể ≠43). Ratchet
  nâng tiếp từ đó. Áp gateway + rà web/workers cùng nguyên tắc.

## Consequences
- (+) Thêm test KHÔNG còn làm tụt gate; coverage = % source thật.
- (−) Floor số đổi (re-based theo source-only) — KHÔNG phải hạ gate, là đổi basis. Cập nhật floors.json.
- Supersede phần measurement-basis của floor T01b (append, không xoá lịch sử).
