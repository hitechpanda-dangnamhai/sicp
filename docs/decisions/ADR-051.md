# ADR-051 — AI Architecture v6 (umbrella, họ ADR)

- **Status:** Accepted
- **Date:** 2026-06-13
- **Refs:** ADR-007 (1 Flask app/8 subgraph), ADR-037 (pattern umbrella tiền lệ), ADR-048 (LangGraph interrupt+resume), ADR-052 (program). Spawn: ADR con khi C7 chạm.
- **Context:** Kiến trúc AI hiện = v5 core đã chạy thật (evidence loop · tool-RAG · confidence deterministic · reflection ×2 · memory 3-tier). Cần 1 ADR đích neo định hướng v6 để các slice cluster C7 cite, tránh refactor graph tuỳ hứng. Chi tiết sống ngoài ADR (Single Home).
- **Decision:** Kiến trúc AI đích = **v5 core + 6 nâng cấp**, mỗi nâng cấp có ADR con khi C7 chạm:
  1. **Planner tách executor**, task-DAG song song — vì latency + decompose.
  2. **Eval-driven:** CẤM refactor prompt/graph khi chưa có golden set + regression gate — vì LLM self-eval không tin được.
  3. **Cost là kiến trúc:** routing theo độ khó + prompt/semantic cache + cost per-tenant.
  4. **Guardrails + per-tool authz tenant-scoped** — biên security mới của multi-tenant AI.
  5. **Tách workload** chat-graph (LangGraph, ADR-048) vs batch/long-running (durable engine, C5/C7).
  6. **Context budget + compaction per-node.**
- **Rationale:** Pattern umbrella theo tiền lệ ADR-037 — neo định hướng 1 nơi, quyết định chi tiết defer tới khi cluster chạm (không over-design sớm). v5 đã tốt; v6 là nâng cấp có chủ đích, không rebuild.
- **Trade-offs:** Giữ ADR-007 (1 app/8 subgraph) — KHÔNG tách microservice AI sớm. 6 nâng cấp đều phụ thuộc cứng eval harness (#2): không thước đo = không refactor.
- **Reference:** `docs/AI_ARCHITECTURE_v6.md` (Single Home chi tiết, 00_CONTEXT §4) — doc này **supersede fossil v5**. Pattern umbrella: ADR-037.
